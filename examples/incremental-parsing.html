2. 使用增量解析 API

支持的操作类型：

操作示例说明

delete { type: 'delete', startLine: 5, deleteCount: 2 } 删除指定行
replace { type: 'replace', startLine: 5, deleteCount: 1, insertLines: ['新'] } 替换行（最常用）
