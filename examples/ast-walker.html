🌲 AST 遍历与查询

📖 示例导航： 快速开始 基础用法 AST 遍历 AST 转换 自定义渲染 增量解析 实时编辑器

使用 walk、findAll、findFirst、getTextContent 操作 AST。

1. walk() — 深度优先遍历

遍历整棵 AST 树，对每个节点执行回调。可以用来收集信息、统计或修改节点。

📝 Markdown 输入
📊 节点类型统计

2. walk() — 提取文档标题目录

遍历 AST 收集所有 heading 节点，用 getTextContent() 获取纯文本内容。

📝 Markdown 输入
📑 提取的目录
层级标题文本

3. findAll() — 查找所有链接和图片

findAll 接受一个类型守卫函数，返回所有匹配节点的数组。

📝 Markdown 输入
🔍 查找结果
类型文本 / AltURL

4. findFirst() — 查找首个匹配节点

找到第一个匹配的节点后立即停止遍历，适用于只需取一个结果的场景。

📝 Markdown 输入
🎯 首个代码块
